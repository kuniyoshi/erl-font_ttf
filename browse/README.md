NAME
====

browse

SUMMARY
=======

This page is for testing.  Can the font show by the browser?

DESCRIPTION
===========

This template have a following work flow.

- .tmpl -> .haml
- .haml -> .html
- .sass -> .css

BROWSE
======

``` bash
  % make
  % open index.html
```

PREREQ
======

- perl-5.10 or later
- Template (perl module)
- some TTF formatted font you have
- ruby
- haml (by gem)
- sass (by gem)
